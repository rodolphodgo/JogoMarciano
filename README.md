## 👽 Caça ao Marciano

Desenvolvido em **Java** que utiliza lógica de busca e persistência de dados para criar uma experiência de "adivinhação" temática, com o objetivo de encontrar em qual árvore o marciano se escondeu dentro de um limite de tentativas. O jogo oferece diferentes níveis de dificuldade e salva os melhores desempenhos em um arquivo local.

## 🎮 Como Jogar

1. **Iniciar:** Escolha a opção `1 - Jogar` no menu.
2. **Dificuldade:** 
   - **Fácil:** 1 a 100 (10 tentativas).
   - **Médio:** 1 a 100 (7 tentativas).
   - **Difícil:** 1 a 200 (7 tentativas).
3. **Dicas:** O sistema avisará se o alvo está em uma árvore de número maior ou menor.
4. **Vitória:** Sua pontuação será calculada e salva no TOP 10.
